# User-CRUD-Project  
A simple front-end only CRUD (Create, Read, Update, Delete) application for managing user data.

## Overview  
This project provides a web interface with pages to:  
- Create a new user (via `create.html`)  
- List existing users (via `index.html`)  
- Update user details (via `update.html`)  

It is built using HTML, CSS, and JavaScript. The current version simulates the CRUD operations purely on the client side.  

## Features  
- Add a new user with required fields  
- View a list of users  
- Update existing user details  
- Delete a user entry  
- Basic styling and layout (CSS folder)  
- Modular JavaScript logic (JS folder)  

## Technologies  
- HTML5  
- CSS3  
- JavaScript (vanilla)  
- No backend/server (client-side only)
